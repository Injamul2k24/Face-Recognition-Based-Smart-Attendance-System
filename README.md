# Face-Recognition-Based-Smart-Attendance-System
Developed an AI-powered smart attendance system using Face Recognition and Computer Vision techniques. Implemented real-time face detection using OpenCV Haar Cascade Classifier and webcam integration. Trained a K-Nearest Neighbors (KNN) model for accurate face recognition and identity prediction. 
# 🎯 Face Recognition Based Smart Attendance System

An AI-powered Smart Attendance System that uses **Face Recognition** and **Computer Vision** to automatically detect and mark student attendance in real time.

Built using **Python, OpenCV, Streamlit, Scikit-learn, NumPy, and Pandas**.

---

# 🚀 Features

✅ Real-time Face Detection using OpenCV  
✅ Face Recognition using KNN Classifier  
✅ Automatic Attendance Marking  
✅ Dynamic CSV Attendance Storage  
✅ Live Attendance Dashboard using Streamlit  
✅ Auto-refresh Attendance Monitoring  
✅ Voice Confirmation for Attendance  
✅ Real-time Webcam Integration  
✅ Student Face Dataset Management

# ⚙️ Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/Face-Recognition-Attendance-System.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd Face-Recognition-Attendance-System
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ How to Run

## Step 1 — Add Face Data

```bash
python data/add_faces.py
```

Enter your name and collect face samples.

Press:

```bash
q
```

to stop capturing.

---

## Step 2 — Run Face Recognition Attendance System

```bash
python data/test.py
```

Press:

```bash
o
```

to mark attendance.

Press:

```bash
q
```

to quit.

---

## Step 3 — Run Streamlit Dashboard

```bash
streamlit run data/app.py
```
